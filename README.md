## 🚀 Running Locally

### Clone the repository:

```bash
git clone https://github.com/MwangiWaraga/portfolio.git
cd portfolio
```

### Install dependencies:

```bash
npm install
```

### Run the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` to view the portfolio.

### Build for production:

```bash
npm run build
```

### Preview the production build:

```bash
npm run preview
```

## 🌐 Deployment

This portfolio is deployed on **GitHub Pages** and can be deployed using:

- **GitHub Pages** (currently used)
- **Vercel**
- **Netlify**
- **Cloudflare Pages**

### GitHub Pages Configuration

For GitHub Pages, the Vite base setting is configured in [vite.config.js](vite.config.js):

```javascript
export default {
  base: '/portfolio/',
};
```
